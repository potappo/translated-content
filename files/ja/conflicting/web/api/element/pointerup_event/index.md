`pointerup` イベントは、ポインターがアクティブではなくなったときに発生します。

## 例

`addEventListener()` を使用した場合

```js
document.addEventListener('pointerup', (event) => {
  console.log('Pointer up');
});
```

`onpointerup` イベントハンドラープロパティを使用した場合

```js
document.onpointerup = (event) => {
  console.log('Pointer up');
};
```
